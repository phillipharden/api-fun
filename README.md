# API Fun 🚀

**API Fun** is a lightweight web application designed to experiment with and demonstrate API integrations.  
The project currently uses the **Spotify API** to search for artists, albums, and songs while showcasing real-time data fetching and dynamic UI updates.

This project was created as a hands-on way to explore working with third-party APIs, asynchronous JavaScript, and responsive front-end development.

---

## 🎯 Project Goals

- Practice working with external APIs
- Build dynamic interfaces that update with live data
- Improve JavaScript async data handling
- Create reusable patterns for future API integrations

---

## ✨ Current Features

- 🎵 Search for **artists, albums, and songs**
- 🔍 Fetch real-time data from the **Spotify API**
- 📀 Display music results dynamically
- ⚡ Asynchronous API requests using `fetch()`
- 🧠 JavaScript-powered DOM updates
- 📱 Fully responsive layout

---

## 🛠 Tech Stack

- **JavaScript** – Application logic and API requests  
- **HTML** – Structure  
- **CSS** – Styling and layout  
- **Spotify API** – Music data integration  

---

## 📦 Installation & Setup

Clone the repository:

```bash
git clone https://github.com/phillipharden/api-fun.git
cd api-fun
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm start
```

The app will run locally at:

```
http://localhost:3000
```
